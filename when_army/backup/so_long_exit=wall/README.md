# so_long

학교 그래픽 라이브러리인 MiniLibX로 만든 간단한 2D 게임입니다.

## Description

so_long은 타일 기반 맵 위에서 플레이어를 움직여, 맵의 모든 수집품을 모은 뒤
출구로 빠져나가는 탑다운 2D 게임입니다. 맵은 `.ber` 파일에서 읽어들여 검증한
뒤, MiniLibX 이미지로 창에 렌더링됩니다.

이 프로젝트의 목표는 C에서 창 관리, 이벤트 처리, 텍스처, 그리고 기본적인 맵
검증 알고리즘을 다뤄보는 것입니다.

**게임 규칙**
- 맵의 모든 수집품(`C`)을 모은 뒤 출구(`E`)에 도달합니다.
- 수집품을 전부 모으기 전까지 출구는 열리지 않습니다.
- 벽(`1`)으로는 이동할 수 없습니다.
- 이동할 때마다 현재 이동 횟수가 셸에 출력됩니다.

## 맵 형식

맵은 다섯 가지 문자로 이루어진 `.ber` 텍스트 파일입니다.

| 문자 | 의미          |
|------|---------------|
| `0`  | 빈 공간        |
| `1`  | 벽            |
| `C`  | 수집품         |
| `E`  | 출구          |
| `P`  | 플레이어 시작   |

유효한 맵의 조건:
- 직사각형이어야 합니다.
- 사방이 벽으로 둘러싸여 있어야 합니다.
- `P`가 정확히 하나, `E`가 정확히 하나, `C`가 하나 이상 있어야 합니다.
- 플레이어가 모든 수집품과 출구에 도달할 수 있는 경로가 존재해야 합니다.

유효하지 않은 맵의 경우 `Error\n`와 짧은 설명을 출력하고 정상 종료합니다.

예시:
```
1111111111111
10010000000C1
1000011111001
1P0011E000001
1111111111111
```

## 조작

| 키              | 동작        |
|-----------------|-------------|
| `W` / `↑`       | 위로 이동    |
| `S` / `↓`       | 아래로 이동  |
| `A` / `←`       | 왼쪽으로 이동 |
| `D` / `→`       | 오른쪽으로 이동 |
| `ESC`           | 종료        |
| 창 닫기 버튼     | 종료        |

## 실행 방법

**빌드**
```
make
```
libft와 MiniLibX를 각자의 Makefile로 컴파일한 뒤 `so_long` 실행 파일을
생성합니다. 사용 가능한 규칙: `all`, `clean`, `fclean`, `re`.

**실행**
```
./so_long map.ber
```
유효한 `.ber` 맵 파일 하나를 인자로 넘깁니다.

## 파일 구성

| 파일            | 역할                                      |
|-----------------|-------------------------------------------|
| `so_long.c`     | 진입점, 초기화, 시작점/크기 탐색           |
| `parsing.c`     | 인자 검사, 파일 읽기, 맵 배열 생성          |
| `validation.c`  | 요소 개수 검사, 직사각형·벽 검사            |
| `dfs.c`         | 맵 복제 및 플러드 필(DFS) 경로 검증         |
| `control.c`     | 키보드 입력과 이동 처리                     |
| `prt_map.c`     | 이미지 로드, 렌더링, 종료 정리              |

## 기술 노트

- 경로 검증은 복제한 맵 위에서 플러드 필(DFS)로 수행합니다. 검증 과정에서
  출구는 벽으로 취급하는데, 이는 "모든 수집품을 모으기 전까지 출구를 지날 수
  없다"는 게임 규칙을 그대로 반영한 것입니다. 덕분에 수집품이 출구를 통해서만
  도달 가능한 잘못된 맵도 걸러낼 수 있습니다.
- 텍스처는 매 프레임 다시 불러오지 않고 시작 시 한 번만 로드해 재사용하여,
  이미지 메모리 누수를 방지합니다.
- 정상 종료와 에러 경로 양쪽 모두에서 힙 할당을 전부 해제합니다.
