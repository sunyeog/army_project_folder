# push_swap

## Description

`push_swap`은 두 개의 스택(`a`, `b`)과 제한된 명령어 집합만으로 정수 목록을
오름차순 정렬하고, 그 정렬을 만들어내는 **가장 짧은 명령어 시퀀스**를 출력하는
프로그램입니다.

- 시작 시 스택 `a`에는 중복 없는 정수들이 들어오고, 스택 `b`는 비어 있습니다.
- 목표는 최소한의 연산으로 `a`를 정렬하는 것이며, 사용할 수 있는 명령어는
  `sa` `sb` `ss` `pa` `pb` `ra` `rb` `rr` `rra` `rrb` `rrr` 뿐입니다.
- 프로그램은 정렬에 필요한 명령어를 한 줄에 하나씩 표준 출력으로 내보냅니다.
- 인자가 정수가 아니거나, `int` 범위를 벗어나거나, 중복이 있으면 표준 에러로
  `Error`를 출력하고 종료합니다.

## Instructions

### 컴파일

```bash
make        # libft 빌드 후 push_swap 생성
make clean  # 오브젝트 파일 삭제
make fclean # 오브젝트 파일 + 실행 파일 삭제
make re     # fclean 후 재빌드
```

### 실행

```bash
# 개별 인자로 전달
./push_swap 2 1 3 6 5 8

# 하나의 문자열로도 전달 가능
./push_swap "2 1 3 6 5 8"
```

정렬이 필요 없는 경우(이미 정렬됨, 인자 없음)에는 아무것도 출력하지 않고
프롬프트로 복귀합니다.

## Technical choices

- **자료구조:** 이중 연결 리스트 기반 스택. `top`/`bottom` 포인터와 `size`를
  들고 다녀 회전(`ra`/`rra`) 연산을 O(1)로 처리합니다. 단방향으로 시작했다가
  역회전(`rra`/`rrb`) 구현을 위해 양방향으로 전환했습니다.
- **파싱:** 모든 인자를 공백 기준으로 분리해 하나의 배열로 합친 뒤, 숫자 검사 →
  `int` 범위 검사 → 중복 검사 → 스택 push 순서로 처리합니다.
- **정렬 알고리즘:**
  - `size` 2·3은 케이스 분기로 직접 정렬.
  - `size` 4 이상은 **인덱스(순위) 변환 후 청크 정렬(chunk sort)**:
    1. 각 값을 크기 순위(0 ~ n-1)로 변환.
    2. 자투리(나머지) 원소를 먼저 `b`로 내림.
    3. 인덱스 구간(청크)별로 작은 순서대로 `b`로 내림.
    4. `b`에서 가장 큰 인덱스부터 찾아, 위치에 따라 `rb`/`rrb`로 회전 후 `pa`.
  - 청크 크기는 입력 규모에 따라 조정(100개 이하 5, 250개 이하 10, 그 이상 14).
